# TinderFE

## Overview
Frontend application for the Tinder-style matching platform. Provides user sign-in, profile browsing, 
swipe left/right functionality, and connection to backend APIs.

## Key Features
- Responsive UI
- Swipe cards interaction
- Connects with backend for auth & data

## Tech Stack
React, Tailwind CSS (or Daisy UI), Vite

## How to Run
1. Install dependencies  
2. Start development server  
3. Connect with backend API
